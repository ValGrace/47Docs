# **App Name**: LughaLink

## Core Features:

- PDF Upload: Upload a PDF document from local storage for translation.
- Language Selection: Select the target Kenyan language (Luo, Kikuyu, Luhya, or Kisii) for translation.
- Translation Engine: Translate the uploaded PDF document into the selected target language, preserving the original document structure, powered by a translation tool that can decide whether to replace parts of the input.
- Download Translated Document: Download the translated PDF document.

## Style Guidelines:

- Primary color: Vibrant green (#A3BE8C), inspired by Kenya's natural environment.
- Background color: Light desaturated green (#E9F5E1), provides a soothing backdrop.
- Accent color: Warm earthy brown (#BF616A), complements the green for a natural, grounding feel.
- Body and headline font: 'PT Sans', a humanist sans-serif suitable for both headlines and body text, provides a modern yet warm aesthetic.
- Use simple, clear icons representing document actions (upload, download, translate) and language options.
- Clean and intuitive layout with clear calls to action. The language selection should be prominently displayed.
- Subtle transitions and loading animations to provide feedback during PDF upload and translation processing.
